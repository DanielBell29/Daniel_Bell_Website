Living Room AV Control System

In my opinion, this project has been the most exciting I’ve worked on. The living room control system, initially set up and programmed by myself using AMX, had become outdated. I was eager to find a sleek solution that would integrate mobile control into the system.

The system comprises a 2.4m projector screen with relay control, a Panasonic projector, a 65" display, and a 4x2 AV switcher, allowing flexible routing between the projector and the 65" display. Previously, each device was controlled via RS-232. To modernize the system, I decided to install a network switch in the rack, connecting it to the house's Wi-Fi to enable wireless control.

After evaluating several options, I chose to build the control interface using Flask. I designed a sleek, scalable user guide that makes adding additional controls straightforward—simply by adding a new button and modal to the homepage.

Unfortunately, I am unable to provide a link to the system, as it runs on a developmental server hosted on a Raspberry Pi. This setup ensures the system is only accessible when connected to the local Wi-Fi network, providing an added layer of security.

Homepage

My goal was to keep the homepage as simple as possible, minimizing the number of options available to the user. I also aimed to make the homepage scalable, ensuring that additional controls—such as lighting control or other features—can easily be added in the future.

Each button on the homepage opens a distinct modal for controlling various system functions.

Power

The power modal provides straightforward control for turning both the projector and the 65" display on and off separately.

The power on/off commands also trigger background actions, including audio level adjustments, unmuting, and video routing, ensuring the system self-adjusts with every power cycle.

Source Selection

This image illustrates how the web app appears on a mobile device. The system is fully flexible and compatible with any device used to access it.

The source selection feature enables users to route any device to either the projector, the TV, or both in real time, with no disruption to the system.

Audio Source Selection

This feature was a recent addition. After the installation and setup were completed, it became clear that the system would benefit from a separate audio routing option. This was particularly important in situations where two users were using the projector and TV independently. One user, for example, may prefer to mute the audio on one device while using the other. This feature proved especially useful when, for instance, a game was installing on the PlayStation, and background music was playing while the user wanted to watch content on the Firestick without interruptions.
